## 📌 Overview
CycleView is a high-fidelity, interactive clinical dashboard designed to visualize comprehensive menstrual cycle data. Built for medical professionals (OB-GYNs, Primary Care Physicians, and Clinical Staff), it translates complex physiological markers—such as Basal Body Temperature (BBT), Luteinizing Hormone (LH) surges, pain, energy levels, and bleeding patterns—into an actionable, thoughtfully designed interface. 

The dashboard enables fast cycle diagnostics following the "7-Second Clinical Scanning Rule" through intuitive color coding, automated AI-driven insights, and pre-computed clinical annotations.

## ✨ Key Features
- **Comprehensive Cycle Visualization:** Interactive line and bar charts tracking BBT, pain variance, energy flux, bleeding intensity, and hormone surges.
- **Automated Clinical Annotations:** Pre-drawn coverlines (e.g., 36.5°C), ovulation window highlights, and multi-marker convergences that minimize diagnostic cognitive load.
- **AI-Assisted Clinical Insights:** Auto-synthesized diagnostics identifying biphasic patterns, abnormal pain spikes, cycle regularity, and peak fertility windows.
- **Medical Dashboard UX & Neumorphic Aesthetics:** Features an F-pattern scanning hierarchy, preattentive color processing for severity, and a low-fatigue neumorphic design optimized for prolonged clinical use.
- **Multi-Metric Correlative Tooltips:** Allows deep dives without changing views, presenting intersecting data points for any specific cycle day simultaneously.
- **Multiple Page Views:** Complete routing between Dashboard Overview, historical cycle data, multi-cycle analytics, and the doctor profile page.

## 🛠 Tech Stack
- **HTML5:** Semantic modular architecture natively optimized for accessibility.
- **CSS3 (Vanilla):** Custom, responsive grid and flexbox layouts utilizing glassmorphism and neumorphic visual aesthetics.
- **JavaScript (Vanilla):** Core application logic, DOM manipulation, multi-page data coordination, and insight rendering.
- **Chart.js & Chart.js Annotation Plugin:** Powering the deeply customizable, reactive data visualization canvases.

## 🚀 Usage 
This project is entirely static and runs securely in the browser with no build step or package dependencies required. 

To view the dashboard, simply open `index.html` in your modern web browser of choice. Enjoy navigating through the overview, analytics, history, and profile pages.
